fuzzyhighlight

Description

This patch make it so that fuzzy matches gets highlighted and is therefore meant to be used together with the patch fuzzymatch.

To make this patch fully integrated with fuzzymatch and support 'case insensitive' features the following line must be changed:

if (*highlight == text[i]) {

into:

if (!fstrncmp(&(*highlight), &text[i], 1)) {
